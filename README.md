# 🌸 Magic Bloom

An interactive Computer Vision project built using **OpenCV** and **MediaPipe** that allows users to create a magical flower bouquet using hand gestures.

The project tracks your hands in real time and responds to different gestures to create an engaging visual experience.

---

## ✨ Features

- 🖐️ Real-time hand tracking using MediaPipe
- 🌸 Interactive flower bouquet animation
- 🌼 Five colorful flowers
  - ❤️ Red
  - 💛 Yellow
  - 🧡 Orange
  - 🩷 Pink
  - 💜 Purple
- 🌿 Animated bouquet with stems and leaves
- 🌺 Flowers bloom smoothly as the right hand opens
- 💡 Brightness of the webcam is controlled using the left hand
- 📷 Real-time webcam processing with OpenCV

---

## 🛠️ Technologies Used

- Python 3
- OpenCV
- MediaPipe
- NumPy

---

## 🎮 Controls

### 🌸 Flower Bloom

Use your **right hand**.

- Keep your palm closed → Flower remains closed.
- Slowly open your palm → Flowers bloom gradually.
- Fully open your palm → Bouquet fully blooms.

---

### 💡 Brightness Control

Use your **left hand**.

- Bring thumb and index finger close together → Decrease brightness.
- Move thumb and index finger apart → Increase brightness.

---

## 💐 Flower Bouquet

The bouquet consists of five flowers:

- ❤️ Red
- 💛 Yellow
- 🧡 Orange
- 🩷 Pink
- 💜 Purple

Each flower blooms smoothly based on the openness of the right hand.
